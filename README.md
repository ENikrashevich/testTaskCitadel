# testTaskCitadel

Реализация мониторинговой системы для операционных систем Windows и Linux (Debian или Ubuntu) На языке C#, с возможностью вывода информации в консоль или файл и настраиваемым периодом отслеживания.

Собираемые метрики

Приложение должно собирать следующие метрики системы:

CPU:
1. Общая загрузка (%).
2. Загрузка по каждому ядру (%).

Память:
1. Всего RAM (MB).
2. Доступно RAM (MB).
3. Использование (%).

Сеть:
1. Входящий трафик (байт/сек).
2. Исходящий трафик (байт/сек)

Технические требования
1. Платформа: .Net 6/7/8/9.
2. Язык: C#.
3. Поддерживаемые OC: Windows или Linux (Ubuntu или Debian).
